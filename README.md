**TO-DO List**

A simple To-Do List application built with React and Vite.
Organize your tasks into To-Do, Doing and Done columns, mark tasks as complete, set priorities like Important, Urgent or Later and move tasks between columns using drag-and-drop to manage your productivity efficiently.

**Features**

- Organize tasks into To-Do, Doing, and Done columns

- Drag-and-drop tasks to move them between columns

- Mark tasks as completed

- Assign priority labels: Important, Urgent, or Later

- Add, edit, and delete tasks easily

- Persistent data using localStorage

- Built with React and Vite

 **To Run**
 
 npm run dev
 
 or
 
 yarn dev
